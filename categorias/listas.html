¿Qué es una lista en HTML?

En HTML, una lista es una forma de mostrar elementos organizados uno debajo del otro.

¿Como funciona una lista en HTML?

HTML solo se encarga de la estructura de la lista, es decir, de organizar los elementos en un formato específico (ordenado o desordenado). La apariencia visual de la lista (como los puntos, números o estilos) se controla mediante CSS.

Comando HTML


              <ul>
                  <li>Manzana</li>
                  <li>Pera</li>
                  <li>Uva</li>
              </ul>

Resultado

  • Manzana
  • Pera
  • Uva

Se usa para mostrar :

  • Opciones de menú
  • Pasos
  • Características
  • Elementos relacionados
