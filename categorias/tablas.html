¿Qué es una tabla en HTML?

Una tabla en HTML es una estructura que permite organizar información en filas y columnas.

Se utiliza cuando necesitamos mostrar datos de manera ordenada y relacionada, como por ejemplo listas de estudiantes, productos, horarios o reportes.

¿Para qué se usa?

  • Mostrar datos organizados
  • Presentar información comparativa
  • Visualizar registros estructurados
  • Exponer resultados o estadísticas
Usar tablas en HTML es emplearlas únicamente para datos estructurados, utilizar etiquetas semánticas como thead y tbody, y asegurar que sean responsivas para dispositivos móviles.

Etiquetas para crear una tabla en HTML

Etiqueta Descripción
<table> Contenedor principal de la tabla
<thead> Agrupa el encabezado
<tbody> Agrupa los datos principales
<tfoot> Agrupa el pie o resumen
<tr> Define una fila
<th> Celda de encabezado
<td> Celda de datos
<caption> Título descriptivo
