Etiquetas HTML en Formularios

Aprende sobre las etiquetas más comunes utilizadas en los formularios HTML. Cada sección incluye ejemplos prácticos y su resultado esperado.

1) <form>

Propósito:

  • Define el formulario y los datos que se enviarán al servidor.
  • Contiene los campos de entrada y los botones de acción.
  • Se utiliza con los atributos action (URL para enviar los datos) y method (método HTTP).

Ejemplo de código:

<form action="/submit" method="POST">
  <input type="text" name="username">
  <button type="submit">Enviar</button>
</form>

2) <input>

Propósito:

  • Crear campos para que el usuario ingrese datos.
  • Puede ser de texto, contraseñas, correos electrónicos, números, etc.

Ejemplo de código:

<input type="text" name="username" placeholder="Nombre">
<input type="password" name="password" placeholder="Contraseña">

3) <textarea>

Propósito:

  • Crear un campo de texto de varias líneas.
  • Ideal para comentarios largos o descripciones.

Ejemplo de código:

<textarea name="comments" rows="4" cols="50">Comentarios</textarea>

4) <label>

Propósito:

  • Asociar una etiqueta con un campo de formulario.
  • Mejora la accesibilidad permitiendo que los usuarios hagan clic en la etiqueta para seleccionar el campo asociado.

Ejemplo de código:

<label for="username">Nombre de Usuario</label>
<input type="text" id="username" name="username">

5) <select>

Propósito:

  • Permite a los usuarios seleccionar una opción de una lista desplegable.
  • Se utiliza junto con <option> para definir las opciones del menú.

Ejemplo de código:

<select name="country">
  <option value="colombia">Colombia</option>
  <option value="mexico">México</option>
</select>

6) <button>

Propósito:

  • Usado para enviar el formulario o ejecutar otras acciones dentro del formulario.
  • Puede ser de tipo `submit`, `reset` o `button`.

Ejemplo de código:

<button type="submit">Enviar</button>

7) <fieldset>

Propósito:

  • Se utiliza para agrupar elementos del formulario, como opciones de un formulario.
  • Mejora la accesibilidad y la organización visual.

Ejemplo de código:

<fieldset>
  <legend>Información Personal</legend>
  <label for="fname">Nombre:</label>
  <input type="text" id="fname" name="fname">
</fieldset>
