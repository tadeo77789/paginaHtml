¿Qué son las etiquetas estructurales?

Las etiquetas estructurales en HTML se utilizan para organizar y definir el contenido de una página web. Estas etiquetas no afectan directamente la apariencia de la página, pero proporcionan una estructura lógica y jerárquica, tanto para los desarrolladores como para los motores de búsqueda y tecnologías de asistencia (como lectores de pantalla).

Algunas de las etiquetas estructurales más comunes son:

header

Define la cabecera de una página o sección, generalmente contiene el título y la navegación.

nav

Define una sección de navegación, conteniendo enlaces a otras partes del sitio o de la misma página, como menús principales o breadcrumbs.

main

Representa el contenido principal de la página. Solo debe haber uno en cada documento.

section

Agrupa contenido relacionado dentro de una página, como artículos o secciones temáticas.

footer

Define el pie de página de la página o de una sección, generalmente contiene derechos de autor y otros enlaces.

aside

Define contenido adicional o secundario, como barras laterales o anuncios.
