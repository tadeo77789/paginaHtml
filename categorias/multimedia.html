Etiquetas HTML para Multimedia

Explora las etiquetas más comunes para trabajar con contenido multimedia en HTML, como imágenes, videos, audios y otros elementos interactivos.

1) <img>

Propósito:

La etiqueta <img> se utiliza para incrustar imágenes en la página web. Permite especificar la fuente de la imagen (atributo src) y su descripción alternativa (atributo alt).

Vista previa:

Ejemplo de código:

<img src="imagen.jpg" alt="Descripción de la imagen" class="img-fluid">

2) <audio>

Propósito:

La etiqueta <audio> se usa para incrustar contenido de audio. Puede incluir controles como reproducir, pausar y volumen mediante el atributo controls.

Vista previa:

Ejemplo de código:

<audio controls>
  <source src="audio.mp3" type="audio/mp3">
  Tu navegador no soporta el elemento de audio.
</audio>

3) <video>

Propósito:

La etiqueta <video> se utiliza para incrustar contenido de video en la página web. Al igual que el audio, puedes agregar controles como play, pausa, y volumen con el atributo controls.

Vista previa:

Ejemplo de código:

<video controls>
  <source src="video.mp4" type="video/mp4">
  Tu navegador no soporta el elemento de video.
</video>

4) <iframe>

Propósito:

La etiqueta <iframe> se usa para incrustar contenido de otras páginas web, como videos de YouTube o mapas de Google.

Vista previa:

Ejemplo de código:

<iframe src="https://www.youtube.com/watch?v=l3HJsXA-Fa4" width="560" height="315"></iframe>
